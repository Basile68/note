# 计算机的三大原则

## 热身问答

硬件物理上存在，软件物理上不存在  
编码是计算机给信息赋予的数字

## 计算机三原则

1. 计算机时执行**输入**、**运算**、**输出**的机器
2. 程序是指令和数据的集合
3. 机器有自己的逻辑

## 输入、运算、输出是硬件的基础

信息输入运算输出，一套流程缺一不可

* 软件是指令和数据的集合
    * 指令即对计算机发出的命令，指令有对应的一或多的名字
* 计算机的逻辑中一切都是数字来表达
* 指令和数据有它们自己的规范

## 热身问答

Hz频率单位，一秒发出一次时钟信号就是1Hz（时钟信号是什么  
Z80 CPU是8比特的CPU

- 比特数=CPU上数据总线的条数/CPU内部参与运算的寄存器的容量

## 时钟信号

- 电子系统中信号是指带有信息的电压或者电流变化
- 每当时钟信号到达特定边沿，电路就会执行一次操作
- 时钟信号就好像带有一个时钟，每隔一定时间就变换一次电压的高低

## 基础知识

1个2进制数（也就是数字IC上1个引脚所能表示的0或者1）所表示的信息称作1比特  
8个2进制数也就是8比特称作1字节  
数字IC的“数字”是指它处理的是离散的二进制信号，而非连续模拟信号（连续模拟信号是指信号的幅值可以在一定范围内连续变化，没有离散的跳跃

# 数据结构

## 热身问答

程序中的变量是指按照变量所储存数据的大小被分配到的一块内存空间，  
把若干个数据沿直线排列起来的数据结构叫做数组，数组的实质是连续分配的一块特定大小的内存空间，    
栈中数据的存取形式是LIFO（last in first out 后进先出）表示优先读取后存入的数据，
队列中数据的存取形式是FIFO（ first in first out 先进先出）表示优先读取先存入的数据。

## 了解内存和变量的关系

对于程序员来说，他们并不需要知道变量被存储到内存空间中的哪个地址上去了，因为当程序运行时是由操作系统为我们从尚未使用的内存空间中划分出一部分分给变量的

## 数组

数组是数据结构的基础，之所以这么说是因为数组反应了内存的物理结构本身  
在内存中存储数据的空间是连续分布的  
在程序中，往往要从内存整体中分配出一块连续的空间以供使用

## 数组的应用

数组是数据结构的基础  
数组是一种直接利用内存物理结构（计算机的特性）的最基本的数据结构  
主要的典型数据结构：

- 栈
    - 把数据像小山一样堆积起来
- 队列
    - 把数据排成一队
- 链表
    - 可以任意地改变数据的排列顺序
- 二叉树
    - 把数据分为两路排列

## 栈

通过使用由数组、栈顶指针以及入栈函数和出栈函数所构成的集合，就能实现栈这种数据结构了。  
应用场景：

- 打开网页，点链接，再点链接……然后按“返回”
- 你点了：百度 → 进入新闻 → 点了一条体育新闻 你现在在第3页。
- 当你点“返回”：先回到“新闻”（第2页）再“返回”，回到“百度”（第1页）
- 就像从一个盒子一张张拿出纸条，顺序是反过来的。
- 浏览器用“栈”记住你去过哪里

栈 = 一个任意大小的数组 + 入栈函数(push) + 出栈函数(pop) + 先入后出
